## Overview

Phys-MAIN is a physics-informed and interpretable Monotone Additive Interaction Network for design-oriented shear capacity prediction and resistance factor calibration of ultra-high-performance concrete (UHPC) beams. The model employs configuration-specific submodels within an adaptive neuro-fuzzy inference system, combining monotone shape functions, bivariate interaction surfaces, and distribution-free conformal prediction intervals, and is calibrated under the ACI 318 LRFD framework at a target reliability index of β<sub>T</sub> = 3.5.

An interactive browser demo is hosted at the project page above.
